## Quick Setup

Добавьте этот workflow, который автоматически создается через `agent-note init`:

```yaml
# .github/workflows/agentnote.yml
name: Agent Note
on:
  pull_request:
    types: [opened, synchronize]
permissions:
  contents: read
  pull-requests: write
jobs:
  report:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
        with:
          fetch-depth: 0
      - uses: wasabeef/AgentNote@v0
```

Этого достаточно. Каждая PR получит AI session report.

## Inputs

| Input | Default | Описание |
|---|---|---|
| `base` | PR base branch | Базовая ветка для сравнения |
| `pr_output` | `description` | Куда выводить PR report: `description`, `comment` или `none` |
| `dashboard` | `false` | Также записывает Dashboard snapshots для текущей PR |

### Публиковать как comment вместо description

```yaml
- uses: wasabeef/AgentNote@v0
  with:
    pr_output: comment
  env:
    GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

### Генерировать Dashboard data вместе с PR report

```yaml
- uses: wasabeef/AgentNote@v0
  id: agent-note
  with:
    pr_output: description
    dashboard: true
  env:
    GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

<Aside type="tip">
  Включайте это только если вам нужен и общий Dashboard. Полная настройка bundle, Pages sync и deploy описана в гайде [Dashboard](./dashboard/).
</Aside>

## Outputs

| Output | Тип | Описание |
|---|---|---|
| `overall_ai_ratio` | number | Общая AI ratio для PR (0-100) |
| `overall_method` | string | `line`, `file`, `mixed` или `none` |
| `tracked_commits` | number | Количество commits с данными agent-note |
| `total_commits` | number | Общее количество commits в PR |
| `total_prompts` | number | Общее количество prompts по всем commits |
| `json` | string | Полный структурированный report |
| `markdown` | string | Сгенерированный markdown report |
| `dashboard_commits` | number | Количество tracked commits, записанных в Dashboard bundle |

### Использование outputs в workflow

```yaml
- uses: wasabeef/AgentNote@v0
  id: agent-note
  with:
    base: main

- run: echo "AI ratio: ${{ steps.agent-note.outputs.overall_ai_ratio }}%"
```

## Report Format

PR report включает:

- **Header** — `Total AI Ratio` с progress bar и отдельной plain-text строкой `Model`
- **Table** — AI ratio по commit, количество prompts и files с attribution
- **Prompts** — сворачиваемый блок с парами prompt / response по commit

<Aside type="tip">
  Хэши commit в report ведут прямо на страницу commit в GitHub.
</Aside>

## Notes Sharing

Action получает `refs/notes/agentnote` с remote. Git hook `pre-push`, установленный через `agent-note init`, автоматически отправляет notes при каждом `git push`, без ручных команд.
