Эта страница показывает команды, которые пользователь запускает напрямую. Для первого setup начните с [Установки](./installation/).

## Частые задачи

- Первый setup: `agent-note init`
- Проверить статус: `agent-note status`
- Посмотреть недавнюю активность: `agent-note log`
- Показать данные commit: `agent-note show`
- Понять, почему изменилась строка: `agent-note why`

<Aside>
  PR Report и Dashboard — это функции GitHub Action. Обычно их создает `agent-note init`, а не внутренние report-команды.
</Aside>

## agent-note init

Настраивает Agent Note в репозитории.

```bash
npx agent-note init --agent claude
```

| Flag | Description |
|---|---|
| `--agent <name...>` | Choose one or more agent adapters: `claude`, `codex`, `cursor`, or `gemini` |
| `--dashboard` | Also create the shared Dashboard workflow |
| `--no-hooks` | Skip agent hook configuration |
| `--no-git-hooks` | Skip git hook installation |
| `--no-action` | Skip workflow creation |
| `--no-notes` | Skip notes auto-fetch config |
| `--hooks` | Only install agent hooks |
| `--action` | Only create workflow |

```bash
npx agent-note status
```

```text
session: a1b2c3d4…
agent:   cursor

```

## agent-note log

Показывает последние commits с AI Ratio.

```bash
npx agent-note log
npx agent-note log 20
```

```text

326a568 test: add tests [a1b2c3d4… | 🤖100% | 1p]
```

Показывает AI Session для commit.

```bash
npx agent-note show
npx agent-note show abc1234
```

## agent-note why

Показывает контекст Agent Note за измененной строкой.

```bash
npx agent-note why src/middleware/auth.ts:42
npx agent-note why src/middleware/auth.ts:42-55
npx agent-note why src/middleware/auth.ts:42:7
npx agent-note why src/middleware/auth.ts#L42
npx agent-note why src/middleware/auth.ts#L42-L55
npx agent-note why src/middleware/auth.ts#L42C7
npx agent-note why @src/middleware/auth.ts#L42
npx agent-note why https://github.com/owner/repo/blob/main/src/middleware/auth.ts#L42
npx agent-note why file:///workspace/repo/src/middleware/auth.ts#L42
npx agent-note why vscode://file/workspace/repo/src/middleware/auth.ts:42:7
```

<Aside>
  `why` принимает `path:line`, `path#Lline`, GitHub file URLs, `file://...`, `vscode://file...` и paths с начальным `@`, скопированные из AI Agent.
</Aside>

## AI Agent Skill

Установите Agent Note Skill, чтобы просить эти задачи на естественном языке.

```bash
gh skill install wasabeef/AgentNote agent-note --agent codex --scope user
```

Выберите подходящее значение `--agent`: `codex`, `claude-code`, `cursor` или `gemini-cli`.
