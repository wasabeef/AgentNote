## Что делает Agent Note

<CardGrid stagger>
  <Card title="Атрибуция ИИ на уровне строк" icon="pencil">
    Алгоритм позиционирования 3-diff определяет, какие строки написаны ИИ, а какие — человеком. Не просто файлы, а конкретные строки.
  </Card>
  <Card title="Отслеживание сессий" icon="document">
    Промпты, ответы, название модели и использованные инструменты — всё привязано к коммитам, которые они породили.
  </Card>
  <Card title="Отчёты в PR" icon="rocket">
    Автоматические отчёты ИИ-сессий в pull request с долей на коммит, количеством строк и связанными хешами.
  </Card>
  <Card title="Нативный Git" icon="setting">
    Хранится в git notes — невидимо для веток и CI. Можно пушить, фетчить и делиться.
  </Card>
</CardGrid>

## Настройка за 60 секунд

<Steps>
1. Запустите init в вашем репозитории

   ```bash
   npx @wasabeef/agentnote init
   ```

2. Закоммитьте сгенерированные файлы

   ```bash
   git add .claude/settings.json .github/workflows/agentnote.yml
   git commit -m "chore: enable agentnote"
   git push
   ```

3. Пишите код с ИИ, коммитьте как обычно — всё отслеживается автоматически
</Steps>
