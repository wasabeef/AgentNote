## Quick Setup

Run this once in your repository:

```bash
npx agent-note init --agent claude
```

That creates the PR report workflow for you:

```yaml
# .github/workflows/agentnote-pr-report.yml
name: Agent Note PR Report
on:
  pull_request:
    types: [opened, reopened, synchronize]
permissions:
  contents: read
  pull-requests: write
jobs:
  report:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v6
        with:
          fetch-depth: 0
      - uses: actions/setup-node@v6
        with:
          node-version: 22
          cache: npm
      - uses: wasabeef/AgentNote@v0
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

That's it. Every PR gets an AI session report.

## Inputs

| Input | Default | 설명 |
|---|---|---|
| `base` | PR base branch | 비교 기준 branch |
| `pr_output` | `description` | PR 리포트의 출력 위치: `description`, `comment`, `none` |

## Dashboard

```bash
npx agent-note init --agent claude --dashboard
```

<Aside type="tip">
  This also adds `.github/workflows/agentnote-dashboard.yml` for the shared Dashboard setup.
</Aside>

## Outputs

| Output | 타입 | 설명 |
|---|---|---|
| `overall_ai_ratio` | number | PR 전체 AI 비율 (0-100) |
| `overall_method` | string | `line`, `file`, `mixed`, `none` |
| `tracked_commits` | number | agent-note 데이터가 있는 commit 수 |
| `total_commits` | number | PR 안의 전체 commit 수 |
| `total_prompts` | number | 모든 commit 의 전체 prompt 수 |
| `json` | string | 전체 구조화 리포트 |
| `markdown` | string | 렌더링된 Markdown 리포트 |

## 리포트 구성

PR 리포트에는 다음이 들어갑니다.

- **Header** — `Total AI Ratio` 와 bar, 그리고 그 아래 `Model`
- **Table** — commit 별 AI 비율, prompt 수, file 목록
- **Prompts** — commit 별 prompt / response 를 접어서 보여주는 섹션

<Aside type="tip">
  리포트 안의 commit hash 는 GitHub 의 commit 페이지로 바로 연결됩니다.
</Aside>
