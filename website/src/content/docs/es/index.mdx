## Qué hace Agent Note

<CardGrid stagger>
  <Card title="Atribución de IA línea por línea" icon="pencil">
    Claude Code obtiene atribución completa por línea. Codex sube a nivel de línea cuando los conteos de patch del transcript coinciden con el commit final.
  </Card>
  <Card title="Seguimiento de sesiones" icon="document">
    Prompts, respuestas, nombre del modelo y uso de herramientas — todo vinculado a los commits que generaron.
  </Card>
  <Card title="Reportes en PR" icon="rocket">
    Reportes automáticos de sesiones de IA en pull requests con proporción por commit, cantidad de líneas y hashes enlazados.
  </Card>
  <Card title="Nativo de Git" icon="setting">
    Almacenado como git notes — invisible para las ramas y CI. Se puede hacer push, fetch y compartir.
  </Card>
</CardGrid>

## Configuración en 60 segundos

<Steps>
1. Ejecuta init en tu repositorio

   ```bash
   npx @wasabeef/agentnote init
   ```

   Para Codex CLI:

   ```bash
   npx @wasabeef/agentnote init --agent codex
   ```

   Para Cursor:

   ```bash
   npx @wasabeef/agentnote init --agent cursor
   ```

2. Haz commit de los archivos generados

   ```bash
   git add .claude/settings.json .github/workflows/agentnote.yml
   git commit -m "chore: enable agentnote"
   git push
   ```

   Los repositorios con Codex deben versionar `.codex/config.toml` y `.codex/hooks.json` en lugar de `.claude/settings.json`.
   Los repositorios con Cursor deben versionar `.cursor/hooks.json`.

3. Claude Code y Codex pueden usar `git commit` normalmente.
   Con los git hooks generados, Cursor preview también puede usar `git commit` normalmente.
   Usa `agentnote commit -m "..."` como respaldo si los git hooks no están disponibles.
</Steps>

## Compatibilidad

| Agente | Estado | Notas |
|---|---|---|
| Claude Code | Soporte completo | Atribución por línea mediante pares de blobs capturados por hooks |
| Codex CLI | Preview | Basado en transcript; sube a nivel de línea cuando los conteos de patch coinciden con el commit |
| Cursor | Preview | Atribución basada en `afterFileEdit` / `afterTabFileEdit`, con subida segura a nivel de línea cuando el blob final aún coincide con la edición de IA |
| Gemini CLI | Próximamente | |
