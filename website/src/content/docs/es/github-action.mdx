## Configuración rápida

Ejecuta esto una vez en tu repositorio:

```bash
npx agent-note init --agent claude
```

Con esto se crea el workflow del informe para PR:

```yaml
# .github/workflows/agentnote-pr-report.yml
name: Agent Note PR Report
on:
  pull_request:
    types: [opened, reopened, synchronize]
permissions:
  contents: read
  pull-requests: write
jobs:
  report:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v6
        with:
          fetch-depth: 0
      - uses: wasabeef/AgentNote@v0
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

Y ya está. Cada PR recibe un informe de sesión de IA.

## Inputs

| Input | Default | Descripción |
|---|---|---|
| `base` | Rama base del PR | Rama contra la que se compara |
| `pr_output` | `description` | Destino del informe del PR: `description`, `comment` o `none` |
| `dashboard` | `false` | Ejecuta el modo Dashboard en lugar del modo PR Report. Normalmente lo genera `agent-note init --dashboard` |

## Dashboard

```bash
npx agent-note init --agent claude --dashboard
```

<Aside type="tip">
  Esto también añade `.github/workflows/agentnote-dashboard.yml` para configurar el Dashboard compartido.
</Aside>

## Outputs

| Output | Tipo | Descripción |
|---|---|---|
| `overall_ai_ratio` | number | AI ratio total del PR (0-100) |
| `overall_method` | string | `line`, `file`, `mixed` o `none` |
| `tracked_commits` | number | Número de commits con datos de agent-note |
| `total_commits` | number | Número total de commits en el PR |
| `total_prompts` | number | Número total de prompts en todos los commits |
| `json` | string | Reporte estructurado completo |
| `markdown` | string | Reporte renderizado en Markdown |
| `should_deploy` | boolean string | Output del modo Dashboard que indica al workflow llamador si debe publicar Pages |

## Qué incluye el reporte

El reporte del PR incluye:

- **Header** — `Total AI Ratio` con barra y, debajo, `Model`
- **Tabla** — AI ratio, cantidad de prompts y archivos por commit
- **Prompts** — entradas `📝 Context`, `🧑 Prompt` y `🤖 Response` por commit en una sección plegable

<Aside type="tip">
  Los hashes de commit del reporte enlazan directamente a la página del commit en GitHub.
</Aside>
