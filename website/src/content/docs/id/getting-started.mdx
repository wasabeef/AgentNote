Agent Note lebih mudah diadopsi ketika bagian-bagiannya jelas: local hooks mengumpulkan context, git notes menempelkan context ke commit, dan GitHub Action mengubahnya menjadi PR Report.

Halaman ini adalah peta. Perintah instalasi ada di halaman berikutnya agar Anda tahu dulu apa yang akan diaktifkan.

## Apa yang akan disiapkan

<CardGrid>
  <Card title="Local capture" icon="laptop">
    Agent hooks mengumpulkan prompt, response, touched files, dan attribution signal saat Anda bekerja.
  </Card>
  <Card title="Git-native history" icon="document">
    Setiap commit mendapat git note di `refs/notes/agentnote`, tanpa mengubah isi branch.
  </Card>
  <Card title="Pull Request context" icon="rocket">
    GitHub Action membaca notes itu dan menulis PR Report untuk reviewer.
  </Card>
  <Card title="Optional Dashboard" icon="setting">
    Jika tim membutuhkan tampilan bersama, Dashboard dapat dipublikasikan ke GitHub Pages.
  </Card>
</CardGrid>

## Alur yang disarankan

<Steps>
1. Baca model singkat ini terlebih dahulu.
2. Ikuti [Instalasi](./installation/) untuk menjalankan `npx agent-note init`.
3. Buka Pull Request dan periksa PR Report.
4. Tambahkan Dashboard hanya jika tim membutuhkan halaman bersama.
</Steps>
