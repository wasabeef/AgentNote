## Apa yang Dilakukan Agent Note

<CardGrid stagger>
  <Card title="Atribusi AI Level Baris" icon="pencil">
    Claude Code mendapat atribusi penuh hingga level baris. Codex naik ke level baris saat jumlah patch di transcript cocok dengan commit final.
  </Card>
  <Card title="Pelacakan Sesi" icon="document">
    Prompt, respons, nama model, dan penggunaan tool — semuanya terhubung ke commit yang dihasilkan.
  </Card>
  <Card title="Laporan PR" icon="rocket">
    Laporan sesi AI otomatis di pull request dengan rasio per commit, jumlah baris, dan hash yang tertaut.
  </Card>
  <Card title="Native Git" icon="setting">
    Tersimpan sebagai git notes — tidak terlihat oleh branch dan CI. Bisa di-push, di-fetch, dan dibagikan.
  </Card>
</CardGrid>

## Setup dalam 60 Detik

<Steps>
1. Jalankan init di repository Anda

   ```bash
   npx agent-note init --agent claude
   ```

   Untuk Codex CLI: `npx agent-note init --agent codex`
   Untuk Cursor: `npx agent-note init --agent cursor`
   Untuk Gemini CLI: `npx agent-note init --agent gemini`
   Beberapa agen sekaligus: `npx agent-note init --agent claude --agent cursor`

2. Commit file yang dihasilkan

   ```bash
   git add .claude/settings.json .github/workflows/agentnote.yml
   git commit -m "chore: enable agent-note"
   git push
   ```

   Repository Codex meng-commit `.codex/config.toml` dan `.codex/hooks.json` sebagai pengganti `.claude/settings.json`.
   Repository Cursor meng-commit `.cursor/hooks.json`.
   Repository Gemini CLI commit `.gemini/settings.json`.

3. Claude Code dan Codex dapat memakai `git commit` seperti biasa.
   Dengan git hook yang dihasilkan, Cursor preview juga bisa memakai `git commit` biasa.
   Gunakan `agent-note commit -m "..."` sebagai fallback bila git hook tidak tersedia.
</Steps>

## Dukungan

| Agen | Status | Catatan |
|---|---|---|
| Claude Code | Dukungan penuh | Atribusi level baris melalui pasangan blob yang ditangkap hook |
| Codex CLI | Preview | Berbasis transcript; naik ke level baris saat jumlah patch cocok dengan commit |
| Cursor | Preview | Atribusi berbasis `afterFileEdit` / `afterTabFileEdit`, dengan upgrade aman ke level baris saat blob final masih cocok dengan edit AI |
| Gemini CLI | Preview | Atribusi file melalui hook `BeforeTool`/`AfterTool` di `.gemini/settings.json` |
