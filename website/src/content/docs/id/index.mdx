## Apa yang Dilakukan Agent Note

<CardGrid stagger>
  <Card title="Atribusi AI Level Baris" icon="pencil">
    Algoritma posisi 3-diff menghitung baris mana yang ditulis AI dan mana yang ditulis manusia — bukan sekadar file mana.
  </Card>
  <Card title="Pelacakan Sesi" icon="document">
    Prompt, respons, nama model, dan penggunaan tool — semuanya terhubung ke commit yang dihasilkan.
  </Card>
  <Card title="Laporan PR" icon="rocket">
    Laporan sesi AI otomatis di pull request dengan rasio per commit, jumlah baris, dan hash yang tertaut.
  </Card>
  <Card title="Native Git" icon="setting">
    Tersimpan sebagai git notes — tidak terlihat oleh branch dan CI. Bisa di-push, di-fetch, dan dibagikan.
  </Card>
</CardGrid>

## Setup dalam 60 Detik

<Steps>
1. Jalankan init di repository Anda

   ```bash
   npx @wasabeef/agentnote init
   ```

2. Commit file yang dihasilkan

   ```bash
   git add .claude/settings.json .github/workflows/agentnote.yml
   git commit -m "chore: enable agentnote"
   git push
   ```

3. Coding dengan AI, commit seperti biasa — semuanya dilacak secara otomatis
</Steps>
