## Quick Setup

Ajoutez ce workflow, créé automatiquement par `agent-note init` :

```yaml
# .github/workflows/agentnote.yml
name: Agent Note
on:
  pull_request:
    types: [opened, synchronize]
permissions:
  contents: read
  pull-requests: write
jobs:
  report:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
        with:
          fetch-depth: 0
      - uses: wasabeef/AgentNote@v0
```

C'est tout. Chaque PR reçoit un AI session report.

## Inputs

| Input | Default | Description |
|---|---|---|
| `base` | PR base branch | Branche de base pour la comparaison |
| `pr_output` | `description` | Destination du PR report : `description`, `comment` ou `none` |
| `dashboard` | `false` | Écrit aussi des Dashboard snapshots pour la PR courante |

### Publier en comment au lieu de description

```yaml
- uses: wasabeef/AgentNote@v0
  with:
    pr_output: comment
  env:
    GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

### Générer des Dashboard data en même temps que le PR report

```yaml
- uses: wasabeef/AgentNote@v0
  id: agent-note
  with:
    pr_output: description
    dashboard: true
  env:
    GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

<Aside type="tip">
  Activez cela seulement si vous voulez aussi le Dashboard partagé. Toute la configuration bundle, Pages sync et deploy est détaillée dans le guide [Dashboard](./dashboard/).
</Aside>

## Outputs

| Output | Type | Description |
|---|---|---|
| `overall_ai_ratio` | number | AI ratio globale de la PR (0-100) |
| `overall_method` | string | `line`, `file`, `mixed` ou `none` |
| `tracked_commits` | number | Nombre de commits avec des données agent-note |
| `total_commits` | number | Nombre total de commits dans la PR |
| `total_prompts` | number | Nombre total de prompts sur tous les commits |
| `json` | string | Rapport structuré complet |
| `markdown` | string | Rapport markdown rendu |
| `dashboard_commits` | number | Nombre de tracked commits écrits dans le Dashboard bundle |

### Utiliser les outputs dans le workflow

```yaml
- uses: wasabeef/AgentNote@v0
  id: agent-note
  with:
    base: main

- run: echo "AI ratio: ${{ steps.agent-note.outputs.overall_ai_ratio }}%"
```

## Report Format

Le PR report contient :

- **Header** — `Total AI Ratio` avec une progress bar, plus une ligne texte séparée `Model`
- **Table** — AI ratio par commit, nombre de prompts et files avec attribution
- **Prompts** — section repliable avec les paires prompt / response par commit

<Aside type="tip">
  Les hashes de commit dans le report pointent directement vers la page GitHub du commit.
</Aside>

## Notes Sharing

L'Action récupère `refs/notes/agentnote` depuis le remote. Le Git hook `pre-push` installé par `agent-note init` pousse automatiquement les notes à chaque `git push`, sans commande manuelle.
