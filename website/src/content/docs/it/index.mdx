## Cosa fa Agent Note

<CardGrid stagger>
  <Card title="Attribuzione IA riga per riga" icon="pencil">
    Un algoritmo di posizionamento 3-diff calcola quali righe sono state scritte dall'IA e quali dall'uomo — non solo quali file.
  </Card>
  <Card title="Tracciamento sessioni" icon="document">
    Prompt, risposte, nome del modello e strumenti utilizzati — tutto collegato ai commit che hanno generato.
  </Card>
  <Card title="Report sulle PR" icon="rocket">
    Report automatici delle sessioni IA sulle pull request con rapporto per commit, conteggio righe e hash collegati.
  </Card>
  <Card title="Nativo Git" icon="setting">
    Archiviato come git notes — invisibile a branch e CI. Condivisibile con il team tramite push e fetch.
  </Card>
</CardGrid>

## Configurazione in 60 secondi

<Steps>
1. Esegui init nel tuo repository

   ```bash
   npx @wasabeef/agentnote init
   ```

2. Committa i file generati

   ```bash
   git add .claude/settings.json .github/workflows/agentnote.yml
   git commit -m "chore: enable agentnote"
   git push
   ```

3. Scrivi codice con l'IA, committa normalmente — tutto viene tracciato in automatico
</Steps>
