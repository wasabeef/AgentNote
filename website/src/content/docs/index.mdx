## What Agent Note Does

<CardGrid stagger>
  <Card title="Line-Level AI Attribution" icon="pencil">
    Claude Code gets full line-level attribution. Codex upgrades to line-level when transcript patch counts match the final commit.
  </Card>
  <Card title="Session Tracking" icon="document">
    Prompts, responses, model name, and tool usage — all linked to the commits they produced.
  </Card>
  <Card title="PR Reports" icon="rocket">
    Automatic AI session reports on pull requests with per-commit ratio, line counts, and linked hashes.
  </Card>
  <Card title="Git Native" icon="setting">
    Stored as git notes — invisible to branches and CI. Pushable, fetchable, shareable.
  </Card>
</CardGrid>

## Setup in 60 Seconds

<Steps>
1. Run init in your repository

   ```bash
   npx @wasabeef/agentnote init
   ```

2. Commit the generated files

   ```bash
   git add .claude/settings.json .github/workflows/agentnote.yml
   git commit -m "chore: enable agentnote"
   git push
   ```

   For Codex CLI, run `npx @wasabeef/agentnote init --agent codex` and commit `.codex/config.toml` plus `.codex/hooks.json` instead of `.claude/settings.json`.

3. Code with AI, commit normally — everything is tracked automatically
</Steps>
