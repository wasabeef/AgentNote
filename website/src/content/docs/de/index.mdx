## Was Agent Note leistet

<CardGrid stagger>
  <Card title="KI-Zuordnung auf Zeilenebene" icon="pencil">
    Claude Code erhält vollständige Zuordnung auf Zeilenebene. Codex wird auf Zeilenebene hochgestuft, wenn die Patch-Zahlen im Transcript zum finalen Commit passen.
  </Card>
  <Card title="Session-Tracking" icon="document">
    Prompts, Antworten, Modellname und Tool-Nutzung — alles verknüpft mit den Commits, die daraus entstanden sind.
  </Card>
  <Card title="PR-Berichte" icon="rocket">
    Automatische KI-Sitzungsberichte in Pull Requests mit Anteil pro Commit, Zeilenanzahl und verlinkten Hashes.
  </Card>
  <Card title="Git-nativ" icon="setting">
    Gespeichert als Git Notes — unsichtbar für Branches und CI. Kann gepusht, gefetcht und im Team geteilt werden.
  </Card>
</CardGrid>

## Einrichtung in 60 Sekunden

<Steps>
1. Führe init in deinem Repository aus

   ```bash
   npx @wasabeef/agentnote init
   ```

   Für Codex CLI:

   ```bash
   npx @wasabeef/agentnote init --agent codex
   ```

   Für Cursor:

   ```bash
   npx @wasabeef/agentnote init --agent cursor
   ```

2. Committe die generierten Dateien

   ```bash
   git add .claude/settings.json .github/workflows/agentnote.yml
   git commit -m "chore: enable agentnote"
   git push
   ```

   Codex-Repositories committen `.codex/config.toml` und `.codex/hooks.json` statt `.claude/settings.json`.
   Cursor-Repositories committen `.cursor/hooks.json`.

3. Claude Code und Codex können `git commit` normal verwenden.
   Mit den erzeugten Git-Hooks kann Cursor Preview auch normales `git commit` verwenden.
   `agentnote commit -m "..."` bleibt der Fallback, wenn Git-Hooks nicht verfügbar sind.
</Steps>

## Unterstützung

| Agent | Status | Hinweise |
|---|---|---|
| Claude Code | Voll unterstützt | Zuordnung auf Zeilenebene über per Hook erfasste Blob-Paare |
| Codex CLI | Preview | Transcript-basiert; Zeilenebene nur, wenn die Patch-Zahlen zum Commit passen |
| Cursor | Preview | `afterFileEdit` / `afterTabFileEdit`-basierte Attribution, mit sicherem Upgrade auf Zeilenebene wenn der finale Blob der KI-Änderung entspricht |
| Gemini CLI | Demnächst | |
