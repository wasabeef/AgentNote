Esta página mostra apenas os comandos que usuários executam diretamente. Para o primeiro setup, comece por [Instalação](./installation/).

## Tarefas comuns

- Primeiro setup: `agent-note init`
- Remover o Agent Note do repositório: `agent-note deinit`
- Verificar estado: `agent-note status`
- Revisar atividade recente: `agent-note log`
- Mostrar dados de um commit: `agent-note show`
- Entender por que uma linha mudou: `agent-note why`

<Aside>
  PR Report e Dashboard são recursos do GitHub Action. `agent-note init` normalmente cria os arquivos de workflow; os relatórios e o Dashboard são gerados depois por esses workflows.
</Aside>

## agent-note init

Configura o Agent Note no repositório.

```bash
npx agent-note init --agent claude
```

| Flag | Descrição |
|---|---|
| `--agent <name...>` | Escolha um ou mais adaptadores de agent: `claude`, `codex`, `cursor` or `gemini` |
| `--dashboard` | Também cria o workflow do Dashboard compartilhado |
| `--no-hooks` | Pula a configuração de hooks do agent |
| `--no-git-hooks` | Pula a instalação de Git Hooks |
| `--no-action` | Pula a criação de workflows |
| `--no-notes` | Pula a configuração de notes auto-fetch |
| `--hooks` | Instala apenas hooks do agent |
| `--action` | Cria apenas workflows |

## agent-note deinit

Remove hooks e config do Agent Note para um ou mais agents.

```bash
npx agent-note deinit --agent claude
```

| Flag | Descrição |
|---|---|
| `--agent <name...>` | `claude`, `codex`, `cursor` or `gemini` |
| `--remove-workflow` | Também remove os workflows de PR Report e Dashboard |
| `--keep-notes` | Mantém a configuração de notes auto-fetch |

## agent-note status

Mostra o tracking state.

```bash
npx agent-note status
```

```text
agent-note v0.x.x

agent:   active (cursor)
capture: cursor(prompt, response, edits, shell)
git:     active (prepare-commit-msg, post-commit, pre-push)
commit:  tracked via git hooks
session: a1b2c3d4…
agent:   cursor
linked:  3/20 recent commits
```

## agent-note log

Lista commits recentes com AI Ratio.

```bash
npx agent-note log
npx agent-note log 20
```

```text
ce941f7 feat: add auth  [a1b2c3d4… | 🤖60% | 2p]
326a568 test: add tests [a1b2c3d4… | 🤖100% | 1p]
ba091be fix: update deps
```

## agent-note show

Mostra a AI Session de um commit.

```bash
npx agent-note show
npx agent-note show abc1234
```


## agent-note why

Mostra o contexto do Agent Note por trás de uma linha alterada.

```bash
npx agent-note why src/middleware/auth.ts:42
npx agent-note why src/middleware/auth.ts:42-55
npx agent-note why src/middleware/auth.ts:42:7
npx agent-note why src/middleware/auth.ts#L42
npx agent-note why src/middleware/auth.ts#L42-L55
npx agent-note why src/middleware/auth.ts#L42C7
npx agent-note why @src/middleware/auth.ts#L42
npx agent-note why https://github.com/owner/repo/blob/main/src/middleware/auth.ts#L42
npx agent-note why file:///workspace/repo/src/middleware/auth.ts#L42
npx agent-note why vscode://file/workspace/repo/src/middleware/auth.ts:42:7
```

<Aside>
  `why` aceita `path:line`, `path#Lline`, GitHub file URLs, `file://...`, `vscode://file...` e paths com `@` inicial copiados de um AI Agent.
</Aside>

## AI Agent Skill

Instale o Agent Note Skill para pedir essas tarefas em linguagem natural.

```bash
gh skill install wasabeef/AgentNote agent-note --agent codex --scope user
```

Para `gh skill install`, escolha o identificador de agent correto: `codex`, `claude-code`, `cursor` or `gemini-cli`.
