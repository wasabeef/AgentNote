## データの流れ

<div class="agent-flow" role="img" aria-label="Agent Note が prompt から git note 共有まで進む流れ">
  <div class="agent-flow__step">
    <span class="agent-flow__label">1. Prompt</span>
    <strong>エージェントに依頼する</strong>
    <p>hook が prompt と turn id を記録します。</p>
  </div>
  <div class="agent-flow__arrow" aria-hidden="true">→</div>
  <div class="agent-flow__step">
    <span class="agent-flow__label">2. Edit</span>
    <strong>AI がファイルを更新する</strong>
    <p>hook または transcript が touched files と attribution signals を残します。</p>
  </div>
  <div class="agent-flow__arrow" aria-hidden="true">→</div>
  <div class="agent-flow__step">
    <span class="agent-flow__label">3. Commit</span>
    <strong>note を記録する</strong>
    <p><code>git commit</code> または <code>agent-note commit</code> で trailer を差し込み、git note を書き込みます。</p>
  </div>
  <div class="agent-flow__arrow" aria-hidden="true">→</div>
  <div class="agent-flow__step">
    <span class="agent-flow__label">4. Share</span>
    <strong>コードと一緒に共有する</strong>
    <p><code>git push</code> で <code>refs/notes/agentnote</code> を remote に送ります。</p>
  </div>
</div>

生成された repo-local git hook が入っていれば、特別なコマンドは不要です。Cursor preview では response hook または local transcript があれば prompt / response を復元でき、git hook が使えない環境では Cursor shell hook が fallback として働きます。

## ストレージ

<CardGrid>
  <Card title="レイヤー 1: ローカル一時データ" icon="laptop">
    `.git/agentnote/sessions/` 内のセッションデータ。プロンプト、変更、blob ハッシュの JSONL ファイル。コミットごとにローテーション。push されません。
  </Card>
  <Card title="レイヤー 2: Git Notes" icon="document">
    `refs/notes/agentnote` の永続レコード。コミットごとに 1 つの JSON。push・fetch・チーム共有が可能。
  </Card>
</CardGrid>

### Note スキーマ

```json
{
  "v": 1,
  "agent": "claude",
  "session_id": "a1b2c3d4-...",
  "model": "claude-sonnet-4-20250514",
  "interactions": [
    {
      "prompt": "JWT 認証ミドルウェアを実装して",
      "response": "ミドルウェアを作成します...",
      "files_touched": ["src/auth.ts"],
      "tools": ["Edit"]
    }
  ],
  "files": [
    { "path": "src/auth.ts", "by_ai": true },
    { "path": "CHANGELOG.md", "by_ai": false }
  ],
  "attribution": {
    "ai_ratio": 73,
    "method": "line",
    "lines": { "ai_added": 146, "total_added": 200, "deleted": 12 }
  }
}
```

<Aside type="tip">
  prompt 一覧はセッション全体ではなく、最終 diff に因果的につながる会話ブロックから選ばれます。file-level の場合も、latest touch の edit と、そこへ至る prompt-only context だけを残します。
</Aside>

<Aside>
  Web の build 出力、Flutter の registrant、Dart codegen、Go protobuf、Rust bindings、Swift / Kotlin の generated source など、代表的な generated artifact は best-effort で `AI ratio` の分母から除外されます。
</Aside>
