## クイックセットアップ

`agentnote init` で自動生成されるワークフローを追加:

```yaml
name: Agent Note
on:
  pull_request:
    types: [opened, synchronize]
permissions:
  contents: read
  pull-requests: write
jobs:
  report:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
        with:
          fetch-depth: 0
      - uses: wasabeef/AgentNote@v0
```

これだけです。すべての PR に AI セッションレポートが投稿されます。

## 入力

| 入力 | デフォルト | 説明 |
|---|---|---|
| `base` | PR のベースブランチ | 比較対象のブランチ |
| `output` | `description` | レポートの出力先: `description` または `comment` |
| `comment` | `true` | レガシー: `false` で投稿を無効化 |

## 出力

| 出力 | 型 | 説明 |
|---|---|---|
| `overall_ai_ratio` | number | PR 全体の AI 比率 (0-100) |
| `overall_method` | string | `line`, `file`, `mixed`, `none` |
| `tracked_commits` | number | agentnote データ付きのコミット数 |
| `total_commits` | number | PR 内の全コミット数 |
| `total_prompts` | number | 全プロンプト数 |
| `json` | string | 構造化レポート全体 |
| `markdown` | string | レンダリング済みのマークダウンレポート |

<Aside type="tip">
  レポート内のコミットハッシュは GitHub のコミットページにリンクされます。
</Aside>
