## Agent Note でできること

<CardGrid stagger>
  <Card title="行単位の AI 著作判定" icon="pencil">
    3-diff アルゴリズムで、AI が書いた行と人間が書いた行を見分けます。ファイル単位ではなく行単位。
  </Card>
  <Card title="セッション記録" icon="document">
    プロンプト・レスポンス・使用モデル・ツール情報をコミットと一緒に記録します。
  </Card>
  <Card title="PR レポート" icon="rocket">
    プルリクエストに AI セッションレポートを自動投稿。コミットごとの AI 比率・変更行数・リンク付きハッシュを表示。
  </Card>
  <Card title="Git ネイティブ" icon="setting">
    git notes で保存するため、ブランチや CI に影響しません。push・fetch でチーム共有も簡単。
  </Card>
</CardGrid>

## 60 秒でセットアップ

<Steps>
1. リポジトリで init を実行

   ```bash
   npx @wasabeef/agentnote init
   ```

2. 生成されたファイルをコミット

   ```bash
   git add .claude/settings.json .github/workflows/agentnote.yml
   git commit -m "chore: enable agentnote"
   git push
   ```

3. あとは普通に AI でコーディングしてコミットするだけ — すべて自動で記録されます
</Steps>
