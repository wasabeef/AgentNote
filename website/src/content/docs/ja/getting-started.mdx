## インストール

<Steps>
1. **init を実行**

   ```bash
   npx @wasabeef/agentnote init
   ```

   Codex CLI の場合:

   ```bash
   npx @wasabeef/agentnote init --agent codex
   ```

   以下が設定されます:
   - エージェントフック (`.claude/settings.json`、または Codex CLI では `.codex/config.toml` と `.codex/hooks.json`)
   - Git フック (`prepare-commit-msg`, `post-commit`, `pre-push`)
   - GitHub Actions ワークフロー (`.github/workflows/agentnote.yml`)
   - notes の自動 fetch 設定

2. **生成ファイルをコミット**

   ```bash
   git add .claude/settings.json .github/workflows/agentnote.yml
   git commit -m "chore: enable agentnote"
   git push
   ```

   Codex CLI では `.codex/config.toml` と `.codex/hooks.json` をコミットします。

3. **完了。** 対応エージェントでコーディングすれば、コミットが自動追跡されます。
</Steps>

## 新しいクローン

各開発者はクローン後に init を実行してローカルの git フックをインストールします:

```bash
git clone <repo> && cd <repo>
npx @wasabeef/agentnote init
```

<Aside>
  Git フックは `.git/` 内に配置されるため、コミットされません。開発者ごとに一度 `init` を実行する必要があります。
</Aside>
