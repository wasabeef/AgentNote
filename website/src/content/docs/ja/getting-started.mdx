## インストール

<Steps>
1. **init を実行**

   ```bash
   npx agent-note init --agent claude
   ```

   Codex CLI の場合:

   ```bash
   npx agent-note init --agent codex
   ```

   Cursor の場合:

   ```bash
   npx agent-note init --agent cursor
   ```

   Gemini CLI の場合:

   ```bash
   npx agent-note init --agent gemini
   ```

   複数エージェントを同時に:

   ```bash
   npx agent-note init --agent claude --agent cursor
   ```

   以下が設定されます:
   - エージェントフック (`.claude/settings.json`、Codex CLI では `.codex/config.toml` と `.codex/hooks.json`、Cursor では `.cursor/hooks.json`、Gemini CLI では `.gemini/settings.json`)
   - Git フック (`prepare-commit-msg`, `post-commit`, `pre-push`)
   - GitHub Actions ワークフロー (`.github/workflows/agentnote.yml`)
   - notes の自動 fetch 設定

2. **生成ファイルをコミット**

   ```bash
   git add .claude/settings.json .github/workflows/agentnote.yml
   git commit -m "chore: enable agent-note"
   git push
   ```

   Codex CLI では `.codex/config.toml` と `.codex/hooks.json` をコミットします。
   Cursor では `.cursor/hooks.json` をコミットします。
   Gemini CLI では `.gemini/settings.json` をコミットします。

3. **完了。** 生成された git hook が入っていれば、Cursor preview を含めて通常の `git commit` で追跡されます。
   git hook が使えない環境では `agent-note commit -m "..."` を fallback として使えます。
</Steps>

## 新しいクローン

各開発者はクローン後に init を実行してローカルの git フックをインストールします:

```bash
git clone <repo> && cd <repo>
npx agent-note init --agent claude
```

<Aside>
  Git フックは `.git/` 内に配置されるため、コミットされません。開発者ごとに一度 `init` を実行する必要があります。
</Aside>

## 自動で行われること

init 後は通常の `git commit` で:

1. **Inject** `Agentnote-Session` trailer をコミットメッセージに追加
2. **Record** prompts、responses、file attribution を git note として保存
3. **Push** `git push` 時に notes も自動で送信

Cursor preview では response hook または local transcript による prompt / response 復元も行われます。
git hook が使えない環境では `agent-note commit` が fallback です。

## 動作確認

```bash
npx agent-note show
```

```
commit:  ce941f7 feat: add JWT auth middleware
session: a1b2c3d4-5678-90ab-cdef-111122223333

ai:      60% (45/75 lines) [█████░░░]
model:   claude-sonnet-4-20250514
agent:   claude
files:   5 changed, 3 by AI

  src/middleware/auth.ts  🤖
  src/types/token.ts  🤖
  CHANGELOG.md  👤
```

```bash
npx agent-note status
```

```text
agent-note v0.x.x

agent:   active (cursor)
capture: cursor(prompt, response, edits, shell)
git:     active (prepare-commit-msg, post-commit, pre-push)
commit:  tracked via git hooks
session: a1b2c3d4…
agent:   cursor
linked:  3/20 recent commits
```

`agent:` は有効な adapter、`capture:` は active な agent hook が何を集めるか、`git:` は managed な repo-local git hook、`commit:` は通常の `git commit` が主経路か、fallback を使うべきかを示します。

## アンインストール

```bash
npx agent-note deinit --agent claude
```

エージェントフックと git フックを除去します。`--remove-workflow` を付けると GitHub Action ワークフローも削除します。
