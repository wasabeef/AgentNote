## Agent Note 能做什麼

<CardGrid stagger>
  <Card title="行級 AI 貢獻度分析" icon="pencil">
    Claude Code 提供完整的行級歸因。Codex 會在 transcript 中的 patch 行數與最終 commit 一致時提升到行級。
  </Card>
  <Card title="工作階段追蹤" icon="document">
    提示詞、回應、模型名稱和工具使用紀錄——全部關聯到對應的提交。
  </Card>
  <Card title="PR 報告" icon="rocket">
    在 Pull Request 自動產生 AI 工作階段報告，包含每次提交的比例、行數和關聯雜湊值。
  </Card>
  <Card title="Git 原生" icon="setting">
    以 git notes 形式儲存——對分支和 CI 不可見。可推送、可拉取、可共享。
  </Card>
</CardGrid>

## 60 秒完成設定

<Steps>
1. 在儲存庫中執行 init

   ```bash
   npx @wasabeef/agentnote init
   ```

   如果你使用 Codex CLI：

   ```bash
   npx @wasabeef/agentnote init --agent codex
   ```

2. 提交產生的檔案

   ```bash
   git add .claude/settings.json .github/workflows/agentnote.yml
   git commit -m "chore: enable agentnote"
   git push
   ```

   Codex 儲存庫需要提交 `.codex/config.toml` 與 `.codex/hooks.json`，而不是 `.claude/settings.json`。

3. 用 AI 寫程式，照常提交——一切自動追蹤
</Steps>
